================================================================================
                         COMANDOS RÁPIDOS
================================================================================

WINDOWS - XAMPP LOCAL:
──────────────────────
Instalar:
  cd C:\xampp\htdocs\vetcare-server\server
  npm install
  npm run setup

Iniciar:
  npm start
  # o
  INICIAR-SERVIDOR.bat

Exportar:
  EXPORTAR-PARA-SERVIDOR.bat

Backup:
  BACKUP-DATOS.bat


LINUX/SERVIDOR:
───────────────
Instalar:
  cd /var/www/vetcare/server
  npm install
  npm run setup

Iniciar (producción):
  pm2 start server.js --name vetcare
  pm2 save
  pm2 startup

Monitorear:
  pm2 status
  pm2 logs vetcare
  pm2 monit

ANTES DE EXPORTAR:
  ☐ Cambiar contraseñas de usuarios demo
  ☐ Eliminar datos de prueba si es necesario
  ☐ Verificar que todo funcione en localhost
  ☐ Hacer backup de database.sqlite
  ☐ Compilar frontend (npm run build)

EXPORTAR:
  ☐ Ejecutar EXPORTAR-PARA-SERVIDOR.bat
  ☐ Verificar que se creó vetcare-para-servidor.zip
  ☐ Copiar database.sqlite (si quieres mantener datos)

SUBIR AL SERVIDOR:
  ☐ Subir ZIP al servidor (scp, FTP, etc)
  ☐ Extraer en /var/www/vetcare/
  ☐ Instalar Node.js en servidor
  ☐ Instalar PM2: npm install -g pm2
  ☐ Instalar dependencias: npm install
  ☐ Copiar database.sqlite (tus datos)
  ☐ Iniciar: pm2 start server.js --name vetcare

CONFIGURAR:
  ☐ Configurar Nginx como proxy
  ☐ Configurar dominio (registro A)
  ☐ Configurar SSL (Certbot)
  ☐ Configurar firewall (ufw)
  ☐ Probar desde móvil/tablet

PRODUCCIÓN:
  ☐ Configurar backups automáticos
  ☐ Documentar accesos y contraseñas
  ☐ Capacitar al personal
